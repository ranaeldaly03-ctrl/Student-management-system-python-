# Student Management System

A desktop application built using Python and Tkinter to manage student records.

## Features
- Add new students
- View all students
- Search students by ID
- Delete selected student
- Clear all student records
- Data stored using SQLite database

## Technologies Used
- Python
- Tkinter (GUI)
- SQLite (Database)

## How it works
Users can enter student information such as name, age, and course.  
The application stores the data in a SQLite database and allows managing the records through a simple graphical interface.
